Tecnologías utilizadas
HTML5 → Estructura de todas las páginas
CSS3 → Estilos personalizados y diseño visual
JavaScript (ES6) → Interactividad y lógica del sitio
Bootstrap 5 → Sistema de grid y componentes responsive
Bootstrap Icons → Iconografía

------------------------------------------------------

Diseño Responsive

El proyecto está desarrollado bajo un enfoque full responsive, garantizando una correcta visualización en:

📱 Dispositivos móviles
📟 Tablets
💻 Desktop / pantallas grandes
Características del diseño responsive:
Uso del sistema Bootstrap Grid
Adaptación automática de secciones según resolución
Menús y navegación optimizados para mobile
Imágenes y contenedores fluidos (%, flex, container-fluid)
Experiencia de usuario consistente en todos los dispositivos

------------------------------------------------------

Arquitectura multiidioma

El sistema multiidioma está implementado por estructura por carpetas, separando cada versión del sitio:

/BR → Portugués (Brasil)
/EN → Inglés
/ → Español (estructura principal)

Cada idioma contiene una copia completa del sitio.

------------------------------------------------------

Funcionalidades principales
🌐 Sitio web multiidioma (ES / EN / BR)
📱 Diseño 100% responsive (mobile, tablet, desktop)
🚛 Sección de servicios logísticos por categoría:
Transporte terrestre
Transporte aéreo
Transporte marítimo
📄 Subservicios detallados por operación
